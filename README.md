# Install

```shell
yarn
```

# Usage

Get your mnemonic. In WarpCast you can get this from Settings > Farcaster Account.

```
export FC_MNEMONIC="your mnemonic here"
```

## Follow everyone @dwr follows (a lot of people)

```shell
yarn run fetch
```

## Unfollow inactive users

```shell
yarn run unfollow
```
